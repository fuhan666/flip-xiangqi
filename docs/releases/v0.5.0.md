# v0.5.0 版本说明

发布日期：2026-04-16

## 版本定位

`v0.5.0` 是翻牌中国象棋的 **真正 3D 对局体验版本**。本次发布将棋盘、棋子与核心交互从“3D 棋盘背景外壳”升级为真正可游玩的 3D 主现法载体，并完成测试、文档与发布验证收口。

## 本次更新

### 1. 真正 3D 主玩法载体

- 棋盘主视图已全面迁移至 Three.js 3D 场景
- 棋子采用圆柱形实体化模型，带有纹理贴图与阴影
- 支持翻牌、移动、吃子、选中的 3D 动画反馈
- 回合切换时相机平滑转到当前行棋方视角

### 2. 3D 交互与动画

- 棋子移动带有弧线轨迹动画
- 翻牌带有旋转动画（暗子翻转至正面）
- 吃子带有爆裂淡出效果
- 选中状态带有脉冲光环

### 3. 性能与构建

- 保留已验证的 vendor chunk 拆分（`three-core` 约 724 kB）
- 首屏仍保持轻量回退与按需加载机制
- `npm run build` 成功，无新增 chunk 警告

### 4. 测试与文档

- 3D 主链路测试全量通过
- README 已更新为 v0.5.0 真 3D 状态描述
- 玩法说明补充当前 3D 交互能力
- 开发规划补充 v0.5 发布收口状态
