# v0.4.0 版本说明

发布日期：2026-04-16

## 版本定位

`v0.4.0` 是翻牌中国象棋的 **3D Beta 发布收口版本**。本次发布不改变核心规则层语义，而是在已并入主线的 Three.js 棋盘外壳基础上，补齐性能缓解、首屏懒加载说明、版本文档与发布验证证据。

## 本次更新

### 1. 构建与包体风险缓解

- 为 Vite 构建加入手动 chunk 拆分
- 将 Three.js 核心运行时与 React Three Fiber 依赖拆分到独立 vendor chunk
- 将 Vite `chunkSizeWarningLimit` 调整到 `750`，用于匹配已知且可接受的 `three-core` 惰性 chunk（约 724 kB）

### 2. 首屏与懒加载体验

- 3D 棋盘继续按需加载，降低首屏阻塞
- 懒加载阶段新增明确的 Beta 文案与加载提示
- 测试环境回退到静态 3D 占位视图，保证自动化验证稳定

### 3. 文档与版本收口

- 项目版本号升级到 `v0.4.0`
- README 已更新为 3D Beta 状态描述
- 玩法说明补充当前 3D Beta 呈现约束
- 开发规划补充 v0.4 Beta 发布收口状态

## 仍未纳入本版

- legal-move / highlighted-targets
- 完整动画系统
- 复杂相机演出
- 联网、AI、账号体系

## 发布验证

建议至少执行以下命令：

```bash
npm test
npm run build
```

预期结果：

- `npm test` 全量通过
- `npm run build` 成功，且默认的 Vite chunk warning 已通过 vendor 拆分 + 阈值校准完成收口

## 审计关注点

- 规则层与 `onCellClick(position)` 协议必须保持不变
- `src/game/*` 不应因本次发布收口产生大范围重写
- 若后续继续打磨 3D 体验，应优先在交互/布局 issue 中推进，而不是在发版 issue 中扩大范围
